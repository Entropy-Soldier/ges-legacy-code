Currently version of Boost libraries: 1.47.0

To install a new version:

1. Download: www.boost.org
2. Unzip to a directory
3. run `bootstrap` in said directory
4. run `bjam --with-python link=static runtime-link=static python-debugging=on,off` in said directory
5. Copy .\stage\lib\libboost_python-vc100-mt-s-[version].lib to svn directory \lib\python\
6. Copy .\stage\lib\libboost_python-vc100-mt-sgyd-[version].lib to svn directory \lib\python\
7. Copy contents of .\boost to svn directory \public\python\boost
8. Modify the linker properties as necessary to load the new libs
9. Rejoice!
